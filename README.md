# Анализ платформы Т-Банк: отели и авиабилеты

Этот проект представляет собой комплексный анализ данных платформы «Т-Банк: отели и авиабилеты», включающий исследовательский анализ данных (EDA) и разработку гипотез по улучшению продукта.

EDA включал:
- Импорт и первичная очистка данных
- Анализ пропущенных значений
- Статистический анализ всех переменных
- Визуализация распределений
- Анализ корреляций между ключевыми показателями
- Сегментация клиентов
- Разработка гипотез
---

## Основное

### Распределение по типам заказов
- **Авиабилеты:** 78.2% заказов (615 тыс.)
- **Бронирование отелей:** 21.8% заказов (171 тыс.)
- Средний чек авиабилетов: **16 000 руб.**
- Средний чек отелей: **12 000 руб.**

### Сегментация по времени бронирования
Рынок покупок поделён на три практически равных сегмента:
- ~40% — Last-Minute покупки
- ~30% — заблаговременное планирование
- ~30% — дальнее планирование

**Ключевой инсайт:** Люди, планирующие заранее, тратят значительно больше и хорошо подходят для Premium-программ, тогда как Last-Minute клиентам нужны промокоды.

### Недоиспользование промокодов
- **С промокодом:** 7,718 заказов (1.3%)
- **Без промокода:** 608,182 заказа (98.7%)

Проблема: недостаточная осведомленность клиентов о возможностях промокодов

## Гипотезы улучшения продукта

### 1️⃣ Промокоды для Last-Minute бронирований
**Цель:** Повышение конверсии  
**Механика:** Промокод снизит психологический барьер при спонтанных покупках

### 2️⃣ Комплексное предложение «Авиа + Отель» с бонусом ⭐
**Цель:** Рост среднего чека  
**Механика:** Персонализированные рекомендации отелей с кэшбэком при комбинированной покупке

**Почему эта гипотеза перспективна:**
- Увеличивает средний чек объединением двух основных продуктов
- Отвечает реальным потребностям: «решить вопрос поездки целиком»
- Премиальные клиенты уже доказали платежеспособность
- Легко масштабируется и тестируется (A/B-тестирование, fake door)

### 3️⃣ Кэшбэк для неактивных пользователей с Premium/Pro подписками
**Цель:** Реактивация платежеспособной аудитории

## Рекомендации по использованию результатов

1. **Для Product Manager:** Приоритизировать гипотезу «Авиа + Отель» как основное направление развития
2. **Для Marketing:** Запустить активную кампанию промокодов для Last-Minute сегмента
3. **Для Analytics:** Настроить метрики для отслеживания успеха гипотез A/B-тестированием
4. **Для Data Science:** Разработать модели персонализации рекомендаций отелей на основе поведения пользователей
